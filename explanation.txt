STEP 3: VPOC Levels Computed from June 17's 5-Min Data
---------------------------------------------------------
The VPOC strategy runs _compute_vpoc() on the previous day's complete 5-min bars:

Algorithm:
  1. Take all 5-min candles from June 17 (history_5min filtered to prev_date)
  2. Divide the day's price range (low to high) into 50 equal price bins
  3. Sum volume for each bin (based on each candle's close price)
  4. VPOC = midpoint of the bin with highest volume (most-traded price of the day)
  5. Value Area = bins that together contain 70% of total day's volume
     VAH = top of Value Area (Value Area High)
     VAL = bottom of Value Area (Value Area Low)

Results from June 17's data:
    VPOC ≈ 1637.5   (the price where most volume traded on June 17)
    VAL  ≈ 1635.10  (bottom of 70% volume zone) ← becomes stop
    VAH  ≈ 1655.06  (top of 70% volume zone)    ← becomes target

These levels represent where institutional participation was concentrated on June 17.
Price tends to respect these levels as support/resistance the following day.

------------------------------------
The VPOC strategy loops over today_5min bars one by one.

At the 09:20–09:25 bar:

    abs(bar.close - vpoc) / vpoc  < 0.002    ← close within 0.2% of VPOC
    bar.close (1636.23) < vpoc (~1637.5)     ← price is below VPOC
    bar.open < bar.close                      ← bar is bullish (rising candle)

This is the "rejection below VPOC" condition:
Price dipped slightly under the most-traded level from June 17 but the bar
closed higher (bullish candle). This means buyers stepped in at the VPOC level —

STEP 3: CPR Levels Computed from June 18's Daily OHLCV
---------------------------------------------------------
The CPR strategy uses the previous day's single daily bar (High, Low, Close) to
compute three pivot levels:

    Pivot = (H + L + C) / 3
    BC    = (H + L) / 2            ← Bottom Central Pivot
    TC    = Pivot + (Pivot - BC)   ← Top Central Pivot

    CPR width     = TC - BC
    CPR width %   = (TC - BC) / Pivot

From June 18's daily bar for CRAFTSMAN:
    Pivot = 9544.50
    BC    = 9542.75   ← becomes stop (if narrow CPR)
    TC    = 9546.25   ← becomes entry (breakout trigger)
    CPR width     = 3.50 points
    CPR width %   = 3.50 / 9544.50 = 0.037%

Narrow CPR threshold = 0.3%.
0.037% is far below 0.3% → this is an extremely NARROW CPR day.

Narrow CPR interpretation: when yesterday's pivot range was very tight, the stock
is coiled and likely to trend strongly in one direction today.

Breakout rule (narrow CPR):
    IF bar.close > TC → buy(TC, TC + 2*(TC-BC), BC)
                        buy(9546.25, 9553.25, 9542.75)
    entry  = TC    = 9546.25
    target = TC + 2*(TC-BC) = 9546.25 + 7.00 = 9553.25
    stop   = BC    = 9542.75
    stop_dist   = TC - BC = 3.50
    target_dist = 2*(TC-BC) = 7.00
